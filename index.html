Solometer by pidder

Low cost Datenlogger/viewer für Wechselrichter von Solaranlagen.

Dieses Projekt macht ein Atmega-Board, wie den Pollin AVR-NetIO oder das Etherrape zu einem kostengünstigen Internet-Datenlogger für die Solaranlage. Zur Zeit wird nur ein Effekta-Wechselrichter unterstützt, die SMA-Software ist in Arbeit. Das Projekt enthält die komplette Software vom Atmega bis zum Datenviewer in Javascript (Der Viewer kommt Mitte Mai 2010).

Dependencies

Beruht auf dem Ethersex Projekt.

Install

Ethersex konfigurieren, übersetzen, flashen und das Board an den WR anschließen. Die Software im Ordner 'server' auf den eigenen PHP-fähigen Webspace hochladen. Fertig.
